# v0.2.3 更新日志

> 发布日期：2026 年 3 月 3 日

v0.2.3 是 v0.2.x 的功能迭代版本，**不包含任何破坏性变更**，所有现有部署和插件无需任何修改即可升级。

## 新增内容

### 新增：文件夹格式插件支持

除原有的 `.dll` 文件方式外，主程序现在支持直接将**插件目录**放入 `plugins/`。插件目录内应包含插件入口程序集及其所有依赖：

```
plugins/
└── MyPlugin/                 ← 文件夹格式插件
    ├── MyPlugin.dll
    ├── MyPlugin.deps.json
    └── SomeDependency.dll
```

文件夹格式适用于本地开发和调试阶段，无需打包直接将 `publish/` 输出目录复制到 `plugins/` 即可。

### 新增：`.sharw` 格式插件支持

主程序现在支持直接加载 `.sharw` 插件包。`.sharw` 是一种 ZIP 压缩包，适用于携带第三方依赖（包括跨平台原生库）的插件分发场景：

```
plugins/
└── MyPlugin.sharw            ← .sharw 格式插件
```

主程序在启动时自动将 `.sharw` 解压到 `plugins/.cache/` 下的对应子目录，并优先使用缓存避免重复解压。

关于 `.sharw` 格式的详细说明与打包方式，请参阅 [.sharw 插件包](/plugin/sharw-package)。
