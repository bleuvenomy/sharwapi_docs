# v0.2.1 更新日志

> 发布日期：2026 年 2 月 26 日

v0.2.1 是 v0.2.x 的补丁版本，重点修复主程序在**自包含 / 单文件发布**场景下的启动崩溃问题。

## 修复内容

### 修复：Serilog 初始化失败导致主程序无法启动

在部分自包含 / 单文件发布产物中，启动时会出现如下错误：

`System.InvalidOperationException: No Serilog:Using configuration section is defined and no Serilog assemblies were found.`

根因是单文件发布模式下，Serilog 无法稳定自动发现 sink 程序集。

本版本已在日志初始化阶段显式注册以下程序集：

- `Serilog.Sinks.Console`
- `Serilog.Sinks.Async`
- `Serilog.Sinks.File`

## 影响

- 修复后，自包含 / 单文件发布包可正常启动
- 现有日志配置结构保持不变，无需修改 `appsettings.json`

## 兼容性说明

- 本版本无新增破坏性变更
- 建议所有 v0.2.0 用户升级到 v0.2.1
