# 从 v0.2.3 升级到 v0.2.4 — 用户侧

本页面面向**部署和运维 SharwAPI 主程序**的用户，介绍从 v0.2.3 升级到 v0.2.4 时需要关注的事项。

v0.2.4 **不包含任何破坏性变更**。直接替换主程序二进制文件即可完成升级，现有插件和配置文件无需做任何调整。

如果你也是插件开发者，请另行阅读 [开发者侧迁移指南](./developer)。

## 破坏性变更

无。

## 升级步骤

1. 用新版主程序替换旧版可执行文件
2. 正常启动，无需其他操作

## 新增能力

无需任何配置变更，现有 `RouteOverride` 配置会在升级后自动触发对应插件的 `OnRoutePrefixResolved` 回调（如果该插件实现了此方法）。
