# 从 v0.2.2 升级到 v0.2.3 — 用户侧

本页面面向**部署和运维 SharwAPI 主程序**的用户，介绍从 v0.2.2 升级到 v0.2.3 时需要关注的事项。

v0.2.3 **不包含任何破坏性变更**。直接替换主程序二进制文件即可完成升级，现有插件无需做任何调整。

如果你也是插件开发者，请另行阅读 [开发者侧迁移指南](./developer)。

## 破坏性变更

无。

## 升级步骤

1. 用新版主程序替换旧版可执行文件
2. 正常启动，无需其他操作

## 新增能力

升级后，`plugins/` 目录额外支持以下两种插件形式：

| 形式 | 示例 | 说明 |
|---|---|---|
| **文件夹** | `plugins/MyPlugin/` | 包含 DLL 及依赖的目录 |
| **.sharw 包** | `plugins/MyPlugin.sharw` | 打包为单文件的 ZIP 压缩包 |

现有插件的存放方式无需更改，升级后会与新格式共存。
